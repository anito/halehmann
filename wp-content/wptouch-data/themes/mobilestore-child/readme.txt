Theme Name: My MobileStore Child
Theme URI: http://www.wptouch.com/
Description: A new child-theme based on MobileStore for HA Lehmann.
Version: 1.0
Depends on: 4.0
Parent: MobileStore
Author: Axel Nitzschner

== Long Description ==

A custom child theme based on WPtouch Pro’s MobileStore theme. By default, all template files will be loaded from MobileStore’s theme directory. To modify a template file, simply copy it into this theme’s directory and modify it accordingly.
